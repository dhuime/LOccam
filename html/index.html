LOccam Version 1.0

LOccam based upon OCCAM langage is a wrapper , writen in C on the top of the RTOS FreeRTOS whose adaptation for Arduino boards has been made in particular by Bill Greiman which one we choose for our project.

LOccam permits a clear structuration of concurrent programs . This is not the case for low level
constructs such semaphore , mutex etc .. , difficult to handle, given by Real Time OS .
Furthermore LOccam is simple to learn and implements very few constructs such : PAR, CHAN, ALT , TIMER etc ..
For that reason LOccam is idealy suited for concurrent programming in the hobbyist world of Arduino.

Whose who are curious of Occam we encourage them to read : Occam 2 Reference Manual .

Installation

Installation of LOccam in the Arduino IDE requires you copy in the libraries folder of your Arduino IDE the following files :

AVR platforms : the files LOccam and FreeRTOS_AVR ( Bill Greiman ) which is the default .

ARM platforms : the files LOccam and FreeRTOS_ARM (Bill Greiman) .
In this later case you must edit the file LOccam.h .
You comment #include < FreeRTOS_AVR > and uncomment #include < FreeRTOS_ARM >.

LOccam comes with many examples who where all tested on the ATMega2560 board.

Warning

New constructs such PRI_ALT and TIMER may be found in this new version of LOccam but who is entirely
compatible with the previous one .
Minor changes in the syntax of alternation processes . readAlt and writeAlt are replaced by readALT( resp writeALT)
Furthermore the implementation of ALT is rebuild for a full compatibility with guards.
Important
For recent versions of the Arduino IDE the function START() must be called START(NULL) if the function TIMER is not used.
